# PokeHub

## Deskripsi Aplikasi
PokeHub adalah aplikasi mobile berbasis web yang dikembangkan menggunakan Flutter. [cite_start]Aplikasi ini dibuat sebagai solusi atas studi kasus dari perusahaan teknologi **Silph Co.** dan developer **Dynamo**[cite: 3, 21]. [cite_start]Sebelumnya, data Pokémon masih berupa file terpisah sehingga pengguna kesulitan mencari informasi yang terorganisir[cite: 9, 10, 11]. 

[cite_start]PokeHub menyajikan data 17 Pokémon secara terstruktur, mudah diakses, menarik, dan informatif sehingga pengguna dapat memahami detail setiap Pokémon secara menyeluruh [cite: 12, 13, 32-250].

## Fitur Utama
1. **Halaman Daftar Pokémon (List View):** Menampilkan daftar singkat seluruh Pokémon agar pengguna dapat mencari dan memilih Pokémon dengan cepat.
2. **Halaman Detail Pokémon (Detail View):** Menyajikan informasi mendalam saat salah satu Pokémon dipilih, meliputi:
   - Identitas (Gambar, Nama, Tipe)
   - Deskripsi Singkat
   - Base Stats (HP, Atk, Def, Sp. Atk, Sp. Def, Speed)
   - Weakness (Kelemahan tipe)
   - Role (Peran dalam tim)
   - Rekomendasi Partner yang cocok
